# Terraform AWS Assignments

This repository contains Terraform assignments completed on AWS.

## Task 1 - EC2 Public Instance
- Created an EC2 instance in a public subnet.
- Configured Security Group for SSH access.
- Used Terraform variables and outputs.

## Task 2 - EC2 Private Instance
- Created an EC2 instance in a private subnet.
- Configured VPC networking.
- Managed infrastructure using Terraform.

## Task 3 - S3 + SQS + Lambda
- Created an S3 bucket.
- Configured SQS queue.
- Created Lambda function.
- Integrated S3 events with Lambda.
- Managed all resources using Terraform.

## Technologies Used
- Terraform
- AWS EC2
- AWS S3
- AWS SQS
- AWS Lambda
